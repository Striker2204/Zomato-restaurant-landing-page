# Zomato Restaurant Landing Page

A responsive Zomato-inspired restaurant landing page built using HTML and CSS. This project showcases restaurant listings, dining categories, ratings, offers, and a modern user interface.

## Features

* Responsive Landing Page
* Restaurant Listings
* Search Bar
* Login & Sign Up Buttons
* Dining Out, Delivery, and Nightlife Categories
* Restaurant Ratings and Offers
* Hover Effects on Cards
* Sticky Navigation Section
* Modern UI Design

## Technologies Used

* HTML5
* CSS3

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `pro1.html` in your browser.
